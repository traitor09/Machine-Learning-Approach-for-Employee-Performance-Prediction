About This Project

This application leverages machine learning to predict employee performance based on various input factors. The predictions help companies assess employee productivity levels and make informed decisions.

Project Goals

  • Develop a robust machine learning model to predict employee performance.
  • Analyze key features that impact productivity.
  • Deploy the model using a Flask API for real-time predictions.

Technologies Used

  • Programming Languages: Python
  • Machine Learning Libraries: scikit-learn, XGBoost
  • Web Framework: Flask
  • Data Visualization: Matplotlib, Seaborn
